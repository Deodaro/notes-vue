# Notes

Приложение для создания заметок. В разработке.

### Используемые технологии
Vue.js

### Установка и запуск

#### Установка
```
npm install
```

#### Запуск в development mode
```
npm run serve
```

### Запуск в production mode
```
npm run build
```
